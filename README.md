# Trancadura WEB
## _Site para liberar o acesso para a sala do MOREA_

### Padrão de Commit:

- Criar branch com seu nome. Ex: invencao, genisson
- Fazer commit para essa branch, com nomes descritivos. Ex: Implementação da funcionalidade da tranca.
